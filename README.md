Title and Purpose:
-------------------
The "Appointment Scheduler" application is designed for scheduling and tracking appointments and customers. It provides a user-friendly interface for managing appointments, users, contacts, and more.

Development Environment:
-------------------------
- IDE: IntelliJ IDEA Community Edition 2023.2.5
- JDK: Java SE 17.0.9
- JavaFX: JavaFX-SDK-17.0.9

Directions for Running the Program:
-------------------------------------
1. Open IntelliJ IDEA.

2. Configure your project settings:
   - Set the JDK to Java SE 17.0.9.
   - Add JavaFX to your project:
     --module-path /path/to/javafx-sdk-17.0.9/lib
     --add-modules javafx.controls,javafx.fxml,javafx.base

3. Import the project into IntelliJ IDEA.

4. Run the application by executing the "main.Main" class.

Additional Report Description:
-------------------------------
My additional report in this application provides information about divisions and their associated counts of customers. This report helps you understand the distribution of customers across different divisions or regions.
